# DreamPop

DreamPop is a bright, friendly dream interpretation video site. It is designed for smooth mobile browsing, rotating dream picks, full dream episode views, comments, YouTube traffic, and a low-fee premium plan.

## Features

- Mobile-first dream video library
- Rotating "Tonight's picks" section
- Full episode overlay with video area and dream explanation
- Comment form per dream
- YouTube channel links
- DreamPop Plus at $0.99/month
- Premium feature concepts: private email interpretation, early access, dream journal vault, priority requests

## Netlify

This is a static site. Publish the project root and leave the build command empty.

```txt
Publish directory: .

```
